# JONTRO Website Documentation

## Overview

This document provides the complete structural and functional documentation for the JONTRO agency website. It outlines the purpose, route, and components of each page in the website architecture.

**Goals of the website:**
- Convert visitors into leads
- Showcase technical capability
- Build trust through case studies
- Generate organic SEO traffic
- Recruit talent
- Clearly present services and pricing

---

## Home Page (`/`)

**Purpose:** Primary conversion page that introduces JONTRO and directs users to services, portfolio, and contact.

**Main Sections:**
- Hero section with headline and CTA
- Stats bar with credibility metrics
- Services preview grid
- Why JONTRO section
- How we work process timeline
- Portfolio highlights
- Testimonials
- Tech stack banner
- Call-to-action strip
- Footer with navigation and socials

---

## Services Page (`/services`)

**Purpose:** Explains all services offered and directs users to detailed service pages.

**Services:**
1. Custom Software Development
2. Mobile App Development
3. AI Agent Development
4. AI Chatbots & Virtual Assistants
5. Agentic Workflow Automation
6. SaaS Product Development
7. API & Microservices Development
8. Business Intelligence Dashboards
9. UI/UX Design & Product Design
10. Cloud Migration & Management

**Each service block includes:**
- Icon
- Service title
- Description
- Feature list
- Industry tags
- CTA button
